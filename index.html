SBB Deep Link Generator

Create dynamic timetable URLs with preserved placeholders (%s) for site-search.

About this Tool

This tool generates SBB deeplinks. You can create URLs for use in site-search applications or for sharing pre-filled timetable searches.

Any field left empty will be omitted from the generated url, and the SBB default will be assumed

It implements the official url parameters published by SBB. For more information, please refer to the SBB Deep Linking documentation.
